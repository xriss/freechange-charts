We currently support historical exchange rates for 190 currencies.

Currencies are expressed as their ISO 4217 codes.

USD to XXX by DAYCSV JSON
USD to XXX by MONTHCSV JSON
USD to XXX by YEARCSV JSON

Data is grabbed from a variety of sources, including IMF, FRED, Code for IATI and OECD.

About

Freechange is a javascript module and data to perform historical currency exchange.
Data conversion is made locally in the browser and never sent to an external server.

Project page, issues and feedback on Github.
